# Blender-Classic-Pudding

A classic pudding scene created in Blender using Shader Nodes.

# Overview

This project is a Blender scene designed to recreate the look of a classic pudding dessert.

It uses Shader Nodes to achieve realistic translucent materials, soft reflections, and warm lighting that highlights the smooth surface of the pudding.

The scene can be freely modified and used for learning Blender material workflows or for creating similar food visualization styles.

# Rendering Settings

Engine: Cycles

Sampling: 1024

Denoising: Enabled

# 概要

このプロジェクトは、クラシックなプリンの質感を再現するためのBlenderシーンです。

シェーダーノードを活用して、半透明素材のリアルな質感・柔らかな反射・プリン表面を引き立てる温かみのあるライティングを表現しています。

Blenderのマテリアル構成の学習や、フードビジュアライゼーション制作のベースとして自由に改変して使用できます。
