# Contributing to Blender-Classic-Pudding

Thank you for your interest in this project!
Here are some guidelines for contributing.

---

## Bug Reports
If you encounter any issues (e.g. rendering errors, missing textures, file corruption), please open an Issue and include:
- Blender version you are using
- OS and hardware specs
- A description of the problem
- Screenshots if possible

## Feature Requests & Suggestions
Have an idea to improve the scene or materials? Feel free to open an Issue with the label `enhancement`.

## Pull Requests
1. Fork this repository
2. Create a new branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add some feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

Please keep changes focused and describe what you modified and why.

## Questions
For general questions, open an Issue with the label `question`.

## Credit
If you share or redistribute a modified version, please include a link to the original repository:
https://github.com/NEO1842/Blender-Classic-Pudding

# Blender-Classic-Pudding への貢献について

このプロジェクトに興味を持っていただきありがとうございます！
貢献する際は以下のガイドラインを参考にしてください。

---

## バグ報告
レンダリングエラー・テクスチャの欠損・ファイルの破損などの問題が発生した場合は、Issueを開いて以下の情報を記載してください：
- 使用しているBlenderのバージョン
- OSおよびハードウェアのスペック
- 問題の詳細な説明
- 可能であればスクリーンショット

## 機能リクエスト・提案
シーンやマテリアルの改善アイデアがあれば、`enhancement` ラベルを付けてIssueを開いてください。

## プルリクエスト
1. このリポジトリをフォークする
2. 新しいブランチを作成する（`git checkout -b feature/your-feature`）
3. 変更をコミットする（`git commit -m 'Add some feature'`）
4. ブランチにプッシュする（`git push origin feature/your-feature`）
5. プルリクエストを開く

変更内容は簡潔にまとめ、何をどのように変更したかを説明してください。

## 質問
一般的な質問は `question` ラベルを付けてIssueを開いてください。

## クレジット
改変版を共有・再配布する場合は、元のリポジトリへのリンクを記載してください：
https://github.com/NEO1842/Blender-Classic-Pudding
